Time Stamp Rename App

撮影日時（Exif）やファイル名の数値を解析し、一括でファイルを整理するツールです。

　アプリの概要
　　本ツールは、写真データから正確な撮影日を抽出することを目的としています。
　　プロパティでは見えるのに取得しづらい、深い階層のデータ（Exif Sub-IFD）まで解析を行うのが特徴です。

　主な機能
　・高度な撮影日抽出
　　　OSのプロパティに表示される撮影日時を正確に取得するため、画像の内部データを詳細にスキャンします。
　・スマートなファイル名解析
　　　データがない場合は、ファイル名に含まれる8桁（年月日）や6桁（年月）の数字から日付を特定します。
　　　※6桁の場合は、月が「01〜12」の範囲内であるかを確認し、誤判定を防ぐ工夫をしております。
　・安全なコピー方式
　　　元ファイルを直接書き換えずに、別フォルダへ新しい名前でコピーを作成するため、安心してお使いいただけます。
　・プレビューと並び替え
　　　実行前に変更後の名前を一覧で確認でき、日付順などへの並べ替えも簡単に行えます。

　使い方
　　1. アプリを起動し、接頭辞や開始番号を入力します。
　　2. 「① フォルダを選択して解析」から対象を選び、一覧の内容を確認します。
　　3. 「② 実行開始」を押すと、新しい名前のファイルが自動で作成されます。

　開発の背景
　　写真の整理において「正しい日付が反映されない」という不便を解消するために開発いたしました。
　　実用性を第一に考え、データの解析精度や使いやすさにこだわって制作しております。
