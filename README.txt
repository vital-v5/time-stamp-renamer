# Time Stamp Rename Tool

撮影日時（EXIF）やファイル作成日を解析し、一括でリネームするツールです。

## 使い方
1. 「TimeStampRenameTool.exe」を起動します。
2. 「接頭辞（例: A）」や「開始番号（例: 0001）」を設定します。
3. 「① フォルダを選択して解析」ボタンを押して対象フォルダを選びます。
4. 一覧で変更後の名前を確認し、問題なければ「② 実行開始」を押します。
   ※「changed」フォルダが自動作成され、そこにコピーが保存されます。

## こだわりポイント
・安心設計：プレビュー機能で実行前に結果を確認でき、元ファイルを上書きしないコピー方式を採用。
・多機能解析：写真（EXIF）の撮影日だけでなく、動画や一般ファイルの作成日・更新日も自動判別してリネームに活用します。
