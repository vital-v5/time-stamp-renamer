Time Stamp Rename App

撮影日時（Exif）やファイル名の数値を解析し、一括でファイルを整理するツールです。

アプリの概要
写真データから正確な撮影日を抽出することを目的としています。
プロパティでは取得しづらい深い階層のデータ（Exif Sub-IFD）まで解析を行うのが特徴です。

主な機能
・高度な撮影日抽出
  画像の内部データを詳細にスキャンし、OSのプロパティに表示される撮影日時を正確に取得します。
・スマートなファイル名解析
  データがない場合は、ファイル名に含まれる8桁（年月日）や6桁（年月）の数字から日付を特定します。
  ※6桁の場合は月が「01-12」の範囲内であるかを確認し、誤判定を防ぎます。
・ドラッグ＆ドロップ対応
  ファイルやフォルダを直接ウィンドウへドロップするだけで、即座に解析が始まります。
・安全なコピー方式
  元ファイルを直接書き換えずに、別フォルダ（changed）へ新しい名前でコピーを作成するため安全です。

使い方
1. 接頭辞や開始番号を入力し、完成イメージを確認します。
2. フォルダやファイルを「選択」または「ドロップ」して解析します。
3. 内容をプレビューで確認し「実行開始」を押すと、新しいファイルが自動作成されます。

動作環境
・OS: Windows 10 / 11
・Python: 3.6 - 3.12
・必要ライブラリ: Pillow、tkinterdnd2
